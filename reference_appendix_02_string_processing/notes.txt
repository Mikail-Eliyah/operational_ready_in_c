=============
with makefile
==============
to compile:
$ make

to clean:
$ make clean

to compile and run:
$ make run


make -npr run print all variables, rules and so on, but not the built-ins.
make -nd run print all decisions.

make -n run shows (but doesn't run) all expected command lines.
